Josephine Tutoring — Brand Assets
==================================

This folder contains the brand files for Josephine's tutoring business.
They match the visual style of josephinethetutor.com.

----------------------------------------------------------------------

LOGO
----
logo-shield.svg
  The shield-with-J logo as a vector file. Use this whenever possible
  (it scales to any size without losing quality). Good for: print
  materials, business cards, anything sized larger than a few hundred
  pixels.

logo-shield-transparent-1024.png
logo-shield-transparent-512.png
  Logo on a transparent background, two sizes. Use these when you need
  a PNG (e.g. uploading to a platform that does not accept SVG).
  Transparent background means the logo will sit on whatever color is
  behind it.

----------------------------------------------------------------------

FACEBOOK BUSINESS PAGE
----------------------
facebook-profile-photo-1024.png
  Square 1024x1024 image with the shield centered on a cream background.
  Upload this as the profile photo (Facebook will display it as a circle,
  but the design accounts for that — the shield stays well within the
  circle's safe area).

facebook-cover-1640x924.png
  The full cover photo at Facebook's recommended upload size. This is
  the file to upload as the cover image. Facebook will crop it
  differently on desktop vs mobile — the shield + tagline + URL are all
  positioned in the center safe-area so they show up on both.

facebook-cover-1200x675.png
  A smaller cropped version (1200x675) in case any platform requires
  that exact size. The same content, just resized.

  When setting up the Facebook business page:
    Page name: Josephine the Tutor (or Josephine Tutoring)
    Category: Tutor / Tutoring Service
    Bio (255 chars): Patient, personalized tutoring in Irvine. From
      toddlers through 12th grade. Academic subjects, test prep,
      creative projects, and college essays. The first 15 minute
      consultation is always free. josephinethetutor.com
    Website: https://josephinethetutor.com
    Phone: 949.393.9913
    Email: josephinethetutor@gmail.com
    Service area: Irvine, CA (and surrounding cities)

EMAIL SIGNATURE
---------------
email-signature.html
  Open this file in Chrome or Safari to see the rendered signature.
  Step-by-step instructions for installing in Gmail and Apple Mail are
  in the file's comments at the top.

  The signature pulls the logo image from josephinethetutor.com, so the
  email client will need internet access to display it. If you ever
  change the logo on the site, the signature updates automatically.
